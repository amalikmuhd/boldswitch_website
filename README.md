# Install

First, install [git](http://git-scm.com/downloads) and the LTS version of [node](https://nodejs.org/). Then:

```sh
git clone https://github.com/OYOLA-LLC/boldswitch_website.git
cd boldswitch_website
npm install
npm start
```
